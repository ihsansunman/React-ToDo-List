## Run Locally

Clone the project

```bash
  git clone https://github.com/ihsansunman/React-ToDo-List
```

Go to the project directory

```bash
  cd React-ToDo-List
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
